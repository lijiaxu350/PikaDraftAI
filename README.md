# PikaDraftAI

PikaDraftAI is a Django web app helping users build competitive Pokémon teams and get strategic feedback from our AI coach, powered by Google's Gemini API.

## Features

- **Pokedex** — Shows all existing Pokémon
- **Teams Dashboard** — Create and manage teams with up to 6 Pokémon
- **EV & Stat Editor** — Fine-tune each Pokémon's EVs and view calculated final stats per slot
- **AI Coach** — Get a team analysis and suggestions from our AI coach
- **User Accounts** — Log in to save your teams

## Tech Stack

- **Backend:** Django
- **Database:** SQLite
- **AI:** Google Gemini API
- **Frontend:** Django templates, CSS
- **Data:** PokeAPI for Pokémon data and sprites
